## Weather App
This project use APIs from https://openweathermap.org/
<br>
You can search weather status of different cities and countries.

LINK: https://weather-app-vercel-zeta.vercel.app/

### Coding technologies used in this project:
<li>Javascript</li>
<li>React</li>
<li>HTML</li>
<li>CSS</li>
<li>Bootstrap</li>
